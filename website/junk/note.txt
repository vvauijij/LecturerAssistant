--------------
создать + активировать виртуальное окружение
python3 -m venv venv
source venv/bin/activate
--------------
качаем все:
pip install -r requirements.txt
--------------
настройка фласка:
export FLASK_APP=app
export FLASK_DEBUG=1
--------------

создать бд:
в командной строке из папки website
flask shell
from app import db
import models
db.create_all()


exit()
--------------
если не работает на маке:
flask run --port 8000




бд и миграции
flask db init
